Recovery

All crypto runs locally. Nothing uploads. Use an incognito window.

Don't linger on a public device. Close this tab when done.

Use a unique passphrase (5+ random words). Do not reuse passwords.

v1 / PBKDF2-HMAC-SHA256 / 1,000,000 iterations / 20-byte key
Salt: v1 + len(name) + name + len(site) + site (NFC, length-prefixed)
TOTP: HMAC-SHA1, 6 digits, 30s (RFC 6238) / Base32 (RFC 4648, no padding)
